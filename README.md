# Jerome Misa — Developer Portfolio

Personal web portfolio for Jerome Misa, BS Information Technology student at San Sebastian College–Recoletos Canlubang.

## 📁 File Structure

```
jerome-portfolio/
├── index.html      ← Main HTML structure & content
├── style.css       ← All styles, layout, animations
├── script.js       ← Scroll reveal, form logic, nav highlight
├── jeromemisa.jpg  ← Profile photo
└── README.md
```
## ✨ Features

- SVG monogram logo (JM) in the navbar
- Dark editorial design with scroll-reveal animations
- Fully responsive (mobile + desktop)
- Skills section with tech logos and short descriptions
- Contact form that opens your mail client pre-filled
- Facebook social link
- Footer year: 2026
- Separate HTML / CSS / JS files for clean organisation
